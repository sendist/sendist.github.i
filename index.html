Lesson Learned

Pelajaran yang bisa saya ambil dari praktikum kali ini adalah tentang pembuatan website. Walaupun hanya website statis dan menggunakan GitHub sebagai hosting ditambah menggunakan template yang sudah dibuat oleh orang lain, tetapi tetap saja sulit. Kesulitan tersebut membuat saya mempelajari banyak hal, mulai dari cara menggunakan GitHub, pembuatan website, dan juga memodifikasi source code html.

Harapan

Setelah lulus dari JTK, saya ingin semua hal yang telah saya pelajari selama di JTK dapat saya terapkan di dalam dunia kerja. Saya berharap pelajaran tersebut dapat membantu karier saya. Selain itu, saya juga berharap supaya ilmu yang saya dapat bisa menjadi bekal untuk melanjutkan pendidikan saya ke S2. Saya masih ingin mempelajari banyak hal di bidang IT karena bidang ini sangatlah luas dan menarik untuk dipelajari.
